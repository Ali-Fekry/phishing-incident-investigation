# Phishing Incident Investigation & Response

## Overview
This project demonstrates the investigation and response to a phishing attack that resulted in user account compromise.

## Scenario
A user reported a suspicious email impersonating a legitimate service. The user clicked the link and submitted credentials, which were later used by an attacker to access the account from an unusual IP address.

## Investigation Steps
- Analyzed phishing email indicators (spoofed sender, malicious link, urgency)
- Reviewed authentication logs for suspicious login activity
- Confirmed account compromise
- Assessed impact and scope of the incident

## Incident Response Actions
- Validated phishing email
- Reset compromised account credentials
- Revoked active sessions
- Recommended security awareness improvements

## MITRE ATT&CK Mapping
- Tactic: Initial Access
- Technique: Phishing (T1566)

## Skills Demonstrated
- Phishing analysis
- Incident response (NIST lifecycle)
- Log analysis
